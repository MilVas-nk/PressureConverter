## Описание

Pressure Converter - это приложение для конвертации единиц измерения давления, разработанное на языке Visual Basic .NET с использованием Windows Forms. Приложение позволяет преобразовывать значения между различными единицами измерения давления и рассчитывать соответствующие значения тока (мА) для калибровки измерительных приборов.

## Функциональность

- **Конвертация давления**: Преобразование между 10 различными единицами измерения давления:
  - Па (Паскали)
  - кПа (килоПаскали)
  - МПа (мегаПаскали)
  - кгс/см² (килограмм-сила на квадратный сантиметр)
  - мбар (миллибар)
  - бар
  - мм вод.ст. (миллиметр водяного столба)
  - мм рт.ст. (миллиметр ртутного столба)
  - атм (атмосфера)
  - psi (фунт-сила на квадратный дюйм)

- **Расчет тока**: Вычисление тока в мА (4-20 мА или 4-20 мА по квадратному корню) на основе значений давления

- **Расчет точек калибровки**: Автоматическое вычисление точек калибровки для 5 и 6 точек в диапазоне

- **Поддержка отрицательных значений**: Возможность работы с отрицательными значениями давления

## Требования

- Microsoft .NET Framework 4.8
- Windows 7 или выше

## Установка

1. Скачайте исполняемый файл из раздела Releases
2. Запустите файл Pressure Converter.exe
3. Приложение готово к использованию

## Использование

1. Введите значение давления в любой из доступных единиц измерения
2. Приложение автоматически отобразит эквивалентные значения во всех остальных единицах
3. Для расчета тока введите минимальное и максимальное значение давления в соответствующие поля
4. Введите текущее значение давления для получения соответствующего значения тока
5. Используйте чекбокс для переключения между линейным и квадратным корневым законом преобразования тока
